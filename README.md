# 交易所界面展示

专业级加密货币交易所完整 UI 展示，覆盖从移动端到 PC 端、Web3 钱包到后台管理的全链路界面设计。

## 产品亮点

### 移动端 App
- **行情 & 交易**：实时行情列表、现货/合约 K 线图、下单面板，极简操作流程
- **现货 & 合约**：支持简易 K 线与深度图切换，交易对选择弹框体验流畅
- **资产管理**：现货账户、合约账户资产一览，支持快速划转
- **Web3 钱包**：内置去中心化钱包，支持 DApp 浏览器、链上兑换、Web3 合约

### PC 端
- **首页**：品牌化营销落地页，行情数据实时滚动展示
- **行情页**：全市场行情列表，涨跌幅、成交量多维排序
- **现货交易**：专业交易界面，深度图、盘口、K 线多窗口布局
- **合约交易**：永续合约完整交易面板，仓位管理、保证金设置一体化
- **账户中心**：现货账户、合约账户资产明细与历史记录
- **下载页**：App 多端下载引导页

### 技术 & 运营
- **API 接口文档**：完整 REST/WebSocket 接口文档，对接友好
- **后台管理系统**：仪表盘数据总览、用户列表管理，满足运营日常需求

---

## 展示站功能

- 按分类筛选：移动端 / PC 端 / 后台 / 其他
- 响应式网格：手机 2 列，平板 3 列，桌面 4-5 列
- 灯箱查看大图，支持键盘导航 / 移动端滑动切换 / 双指缩放拖动
- Binance 暗色设计风格

## 启动

```bash
npm install
npm run dev
```

局域网其他设备可通过终端输出的 **Network** 地址访问。

## 构建

```bash
npm run build
```
